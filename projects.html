Human Activity Recognition by Spark

Given a large amount of data (15630426 records), the whole computing process was conducted based on distributed setting using PySpark. The data retrieval, exploratory data analysis (EDA) was conducted using Spark SQL. Feature engineering and machine learning algorithms were implemented using Spark ML. The entire project relies on AWS - the data was stored in S3 and computing was performed on EMR.

Topic Modeling for Jeopardy! Questions

Jeopardy! is a game show in which contestants compete to answer questions the fastest against one another. Questions are placed on a game board under themes and monetary value. This project identifies and classifies the topics of Jeopardy! question by topic modeling methods, including Non-Negative Matrix Factorization (NMF), Latent Dirichlet allocation (LDA), and word embedding with k-means clustering

Ames House Prices Prediction

Given Ames Housing dataset, a mixed stepwise selection was performed to reduce the set of variables and select the best model based on AIC, BIC, and adjust R-squared. With the best model selected, the model assumptions were checked regarding normality and several solutions were proposed to solve the assumption violation. The model was then tested on unseen data and scored on Root-Mean-Squared-Error (RMSE).
